---------------------- Start Up settings ----------------------
1- python -m venv datilio_venv  (for Linux: python3 -m venv datilio_venv)
2- .\datilio_venv\Scripts\activate (for Linux: source datilio_venv/bin/activate)
3- pip install -r requirements.txt
3.1- pip install ipykernel   ==> for testing codes by jupyter-notebook
4- Create a ".env" file next to the main.py file, then copy and paste the 'Environment Variables' from the section below, changing their values based on your settings.
5- python main.py (If you want to run in debug mode, press `Ctrl+Shift+D` and then `Enter`.)

Note: in .vscode>launch.json add --port 8000 for local test as follow:
        "configurations": [
                {
                    "name": "Python Debugger: FastAPI",
                    "type": "debugpy",
                    "request": "launch",
                    "module": "uvicorn",
                    "args": [
                        "main:app",
                        "--reload",
                        "--port",
                        "8000"
                    ],
                    "jinja": true
                }
            ]
